Mesh¶

Functions for working with CINEMA 4D’s point and polygon objects.

TogglePolySelection(obj)¶
SelectAllPolys(obj)¶
GetSelectedPoints(obj)¶

Returns list of selected point indices.

To get the actual point(s), using an index from the returned list do something like

allpoints = op.GetAllPoints()
point = allpoints[index]
GetSelectedPolys(obj)¶

Returns list of selected polygons indices.

To get the actual polygon(s), using an index from the returned list do something like

allpolys = obj.GetAllPolygons()
poly = allpolys[index]
CalcPolyCentroid(p, obj)¶

Calculate the centroid of a polygon by averaging its vertices.

CalcPolyNormal(p, obj)¶

Calculate the orientation of face normal using Newell’s method.

See CalcVertexNormal() for an example of usage within the calling context.

CalcVertexNormal(v, idx, obj)¶

Calculate the vertex normal by averaging surrounding face normals.

Usually called from a construct like the following:

for i, point in enumerate(obj.GetAllPoints()):
   vn = CalcVertexNormal(point, i, obj)
CalcThreePointNormal(a, b, c)¶

Calculate the surface normal of a three point plane.

Doesn’t take orientation of neighboring polygons into account.

CalcTriangleArea(p, obj)¶

Calculate area of a triangle using |(v3 - v1) x (v3 - v2)|/2.

CalcPolyArea(p, obj, normalized=False)¶

Calculate the area of a planar polygon.

CalcBBox(e, sel_only=False)¶

Construct a BBox for a c4d.PointObject or a c4d.CPolygon.

Note that if you are interested in the midpoint or radius only, you can use the built-in c4d.BaseObject.GetMp() and GetRad() methods respectively.

Parameters:sel_only (bool) – if True and e is a c4d.PointObject, use selected points only. Otherwise use all points.
CalcGravityCenter(obj)¶

Calculate the center of gravity for obj.
